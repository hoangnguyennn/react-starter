## React Starter

### Commands

Run dev:

```bash
$ pnpm dev
```

Run build:

```bash
$ pnpm build
```

Run linting:

```bash
$ pnpm lint # show eslint errors

$ pnpm lint:fix # fix eslint errors
```

### Features

- routing
- layouts, pages and components
- axios
- multi languages
- custom hooks
- storybook
- unit test (includes mock API test)
- TypeScript
